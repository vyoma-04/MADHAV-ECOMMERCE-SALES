# 📊 MADHAV E-COMMERCE SALES DASHBOARD

## 📝 Project Description

This project presents an **interactive E-commerce Sales Dashboard** built to analyze and visualize sales performance data. The dashboard provides meaningful insights into **sales, profit, customer behavior, and product performance** using a visually appealing and user-friendly interface.

It helps businesses understand trends, identify profitable areas, and make data-driven decisions.

---

## 🚀 Key Highlights

* Designed a **professional dashboard UI** with a modern gradient theme
* Integrated **multiple KPIs** for quick business insights
* Visualized **sales, profit, quantity, and AOV (Average Order Value)**
* Added **filters (Quarter selection)** for dynamic data analysis
* Included **category-wise, state-wise, and payment-wise breakdowns**

---

## 📌 Features

### 🔢 KPI Metrics

* **Total Sales Amount**
* **Total Quantity Sold**
* **Total Profit (including loss tracking)**
* **Average Order Value (AOV)**

---

### 📊 Visualizations

* 📈 **Profit by Month**
  Shows monthly performance trends (profit & loss)

* 🧾 **Quantity by Category**
  Displays distribution across:

  * Clothing
  * Electronics
  * Furniture

* 🗺️ **Sales by State**
  Highlights top-performing states:

  * Madhya Pradesh
  * Maharashtra
  * Uttar Pradesh
  * Kerala

* 💳 **Payment Mode Analysis**

  * COD (Cash on Delivery)
  * UPI
  * Credit Card
  * Debit Card

* 📦 **Profit by Sub-Category**
  Identifies best and worst performing products like:

  * Bookcases
  * Tables
  * Accessories
  * Saree
  * Electronic Games

---

### 🎛️ Filters

* Quarter-wise filtering (Q1, Q2, Q3, Q4)
* Dropdown selection for dynamic analysis
